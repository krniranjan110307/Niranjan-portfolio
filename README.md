# Niranjan Portfolio

A premium, responsive portfolio built with React, TypeScript, Vite, and Tailwind CSS.

## Tech

- React
- TypeScript
- Tailwind CSS
- Vite

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design direction

- Soft and light premium palette
- Glassmorphism panels
- Responsive layout across mobile, tablet, and desktop
- Subtle floating and shimmer animations
- Typed content structure for easy future edits
